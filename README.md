# node-random-number

Tools ini digunakan untuk membuat nomor WhatsApp secara Random,
***Cocok untuk Apakah Tools ini?***
1. Untuk menyebar Iklan seperti Judi Online, Jualan Online, Dan Lainnya ke nomor Random
2. Mencari teman dan Lainnya

## Cara Instalasi 
***Git clone**
```bash
git clone https://github.com/NekomonHub/node-random-number
```
***Penggunaan Lengkap***
```bash
cd node-random-number
```
```bash
npm install
```
```bash
npm start
```
**Atau juga memakai**
```bash
node main.js
```
